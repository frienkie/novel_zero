# 介绍&前言

庄司熏，1937年出生，本名福田章二。1957年进入东京大学文科就读。1958年发表处女作《丧失》，获得中央公论新人赏，但当时的评委三岛由纪夫和江藤淳并不认可他。后来于1960年开始经过6年罢笔，1966年在“60会“集社发表数十篇短篇小说。在东大斗争的1969年发表了《小红巾，要小心》，并以此获得芥川赏，并得到三岛由纪夫的认可。后续经历不谈，但他现在还活着，所以此小说仍然受到版权保护，并从青空文库下架。

小红巾指是赤頭巾ちゃん，也就是当时参加左翼运动的学生的代称。不过这部作品并没有过多涉及政治内容，而是讲述了高三学生“薰”因为东大斗争的缘故，当年的东大入试取消而郁闷和百无聊奈的生活。

因为本文风格和《麦田里的守望者》高度相似，故而庄司熏也被称为“日本的塞林格”。这是我一直想要翻译的作品，所以我打算3年到5年的时间翻译完成，拿出终稿。
